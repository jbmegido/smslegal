## VI. Proceso de Activación

### Flujo operativo

1. **Solicitud:** El cliente completa el formulario de acceso con declaración de responsabilidad.

2. **Evaluación:** El equipo técnico revisa el perfil y el modelo de negocio. No se admiten todos los solicitantes.

3. **Documentación:** El cliente debe aportar:
   - CIF/NIF de la empresa o autónomo
   - Modelo de formulario de consentimiento utilizado
   - Declaración jurada de titularidad legítima de la base de datos

4. **Configuración:** Tras aprobación y pago de la configuración inicial (150 €), se habilita el panel de control con credenciales de acceso.

5. **Formación:** El cliente recibe un manual operativo en PDF. No se realizan sesiones en vídeo ni llamadas de onboarding.

### Tiempos estimados

- Evaluación inicial: 3-5 días laborables
- Configuración técnica: 2-3 días laborables tras recepción de pago
- Primera campaña: cliente puede enviarla inmediatamente tras carga de base de datos

**No se garantizan plazos.** Si el servicio no es adecuado para su caso de uso, se rechaza la solicitud sin obligación de justificación.

### Política de admisión

Este es un servicio de infraestructura técnica, no una plataforma de marketing abierta. Se reserva el derecho de admisión basándose en:

- Evaluación de riesgo legal
- Capacidad operativa
- Coherencia con el modelo de negocio del proveedor

**Si su caso de uso requiere personalización, integración API compleja o asesoramiento legal, este servicio NO es adecuado para usted.**
