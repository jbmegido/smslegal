## III. Criterios de Admisión

Este servicio está diseñado exclusivamente para negocios locales con bases de datos propias de clientes que han dado consentimiento explícito.

### ✓ Perfiles admitidos

- Restaurantes y cafeterías con programa de fidelización
- Clínicas dentales, fisioterapeutas, centros médicos
- Gimnasios, centros deportivos, estudios de yoga
- Talleres mecánicos con histórico de clientes
- Peluquerías y centros de estética
- Inmobiliarias con cartera propia de clientes
- Tiendas de barrio con programa de socios
- Academias y centros de formación

**Condición obligatoria:** El cliente debe haber obtenido el consentimiento mediante:

- Formulario físico firmado con casilla específica para SMS
- Formulario web con checkbox desglosado (no pre-marcado)
- Registro telefónico con grabación y doble confirmación
- Cualquier otro método que cumpla el estándar RGPD

### ✗ Perfiles NO admitidos

- Empresas con listas compradas o alquiladas
- Bases de datos obtenidas por scraping web o redes sociales
- Contactos sin consentimiento específico para SMS comercial
- Sectores regulados sin acreditación (farmacia, banca, seguros)
- Campañas políticas o de captación sindical
- Promoción de servicios financieros no regulados (criptomonedas, forex)
- Contenido adulto, juego o apuestas
- Multinivel, pirámides o esquemas similares

### Verificación obligatoria

Antes de la activación, el cliente debe:

1. Declarar por escrito ser titular legítimo de la base de datos
2. Confirmar haber obtenido consentimiento válido según RGPD
3. Aportar ejemplo de formulario de captación de consentimiento
4. Aceptar auditoría documental en caso de denuncia o investigación AEPD

**El proveedor se reserva el derecho de rechazar clientes sin justificación.**
