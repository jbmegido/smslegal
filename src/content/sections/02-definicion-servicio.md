## II. Definición del Servicio

SMS Infrastructure es un servicio técnico de envío masivo de SMS con funcionalidades de compliance RGPD/LSSI para negocios locales españoles.

### El servicio INCLUYE

- Envío programado de SMS a listas propias del cliente
- Verificación automática contra Lista Robinson nacional
- Gestión automática de palabras STOP/BAJA
- Panel de trazabilidad con registro temporal de consentimientos
- Almacenamiento cifrado de bases de datos durante 6 años
- Soporte técnico por email (respuesta en 48h laborables)
- Dashboard web con estadísticas básicas

### El servicio NO INCLUYE

- Asesoramiento jurídico sobre compliance
- Auditoría de legitimidad de bases de datos
- Compra o alquiler de listas de contactos
- Personalización de funcionalidades
- Integración API personalizada
- Llamadas telefónicas de soporte
- Garantía de tasas de entrega específicas
- Campañas de marketing diseñadas por nosotros

### Responsabilidad delimitada

El proveedor NO es responsable de:

1. La legalidad del origen de las bases de datos del cliente
2. La correcta obtención de consentimientos previos
3. El contenido de los mensajes enviados
4. El cumplimiento sectorial específico (sanidad, finanzas, etc.)
5. Las consecuencias legales del uso indebido del servicio
